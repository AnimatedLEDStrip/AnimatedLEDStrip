ReqLevel

internal enum class ReqLevel

Helper enum for specifying the requirement level of an animation parameter.

Enum Values

REQUIRED

Animation parameter must be set by the user

OPTIONAL

Animation parameter may be set by the user, otherwise will be set to a default as specified (color# will default to animatedledstrip.colors.ccpresets.CCBlack)

NOTUSED

Animation does not use parameter
