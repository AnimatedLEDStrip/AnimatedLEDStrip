Direction

enum class Direction

Enum that indicates if an animation should appear to move 'Forward' or 'Backward'.

The start of the LED strip is the end where the signal is connected (small arrows on strip pointing away from it). The end of the LED strip is the opposite end from the start (small arrows on strip point towards it).

Enum Values

FORWARD

Animation appears to move from the start to the end of the strip/section.

BACKWARD

Animation appears to move from the end to the start of the strip/section.
