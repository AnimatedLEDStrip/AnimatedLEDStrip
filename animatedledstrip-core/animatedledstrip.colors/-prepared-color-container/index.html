PreparedColorContainer

class PreparedColorContainer : ColorContainerInterface

A prepared ColorContainer that holds a set of colors that blend from one to the next. This is created by calling the prepare() function on a ColorContainer. Used by methods that set the color of pixel(s) on a strip.
