sparkle

private val sparkle: (AnimationData) -> Unit

Runs a Sparkle animation.

Each LED is changed to pCols[0] for delay milliseconds before reverting to its original color. A separate thread is created for each pixel. Each thread saves its pixel's original color, then waits for 0-5 seconds before sparkling its pixel.
