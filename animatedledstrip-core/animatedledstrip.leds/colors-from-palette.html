colorsFromPalette

fun colorsFromPalette(palette: List<ColorContainer>, numLEDs: Int): Map<Int, ColorContainer>

Create a collection of colors that blend between multiple colors along a 'strip'.

The palette colors are spread out along the strip at approximately equal intervals. All pixels between these 'pure' pixels are a blend between the colors of the two nearest pure pixels. The blend ratio is determined by the location of the pixel relative to the nearest pure pixels.

Parameters

palette - A list of ColorContainers used to create the collection's colors

numLEDs - The number of LEDs to create colors for

Return
A Map<Int, ColorContainer> with each pixel index mapped to a ColorContainer
